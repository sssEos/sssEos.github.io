R Package

outlierfix (Documentation)

outlierfix is an R package I developed to support empirical analysis. It helps researchers detect and manage outliers in continuous variables, making data-cleaning decisions more systematic, transparent, and easier to document across research projects. Jump right in — install it with the commands below! Feedback and suggestions are always welcome! 😊 

# install.packages("devtools")

library(devtools)

install_github("sssEos/outlierfix")

library(outlierfix)
