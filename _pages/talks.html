R Package

[Outlierfix](https://rdrr.io/github/sssEos/outlierfix/man/fixout_conti.html)

The R package has been developed for outlier detection and addressing them appropriately for continuous variables. This process is an integral part of data cleaning and we put together this package to ease that pain for fellow colleagues. The package can be downloaded using the following commands (comments/feedbacks are welcome): #install.packages(devtools)
library(devtools)
install_github("sssEos/outlierfix")
library(outlierfix)
